휴고엔 강력한 메뉴 시스템이 있습니다. 이 메뉴시스템으론 다음과 같은 일을 할 수 있습니다.

  • 제한없는 메뉴 depth
  • 어떠한 컨텐츠도 필요없는 메뉴 엔트리 만들기
  • 엑티브 요소 구별

https://gohugo.io/content-management/menus/

What is a Menu in Hugo?

menu는 .Site.Menus 사이트 변수로 사용 가능한 메뉴 엔트리들의 배열입니다. main이라는 메뉴가 있으면 .Site.Menus.main 으로 접근 가능합니다.

Add content to menus

휴고에선 front matter를 통해 메뉴에 컨텐츠를 집어넣는 것이 가능합니다.

만약 메뉴에 엔트리만 추가하는 것이 목적이라면 다음과 같이 간단합니다.

A Single Menu

    name = "getting started"
    post = "<span class='alert'>New!</span>"

    weight = -100

Nesting

컨텐츠의 모든 중첩은 parent 필드를 통해 이루어집니다.

엔트리의 parent는 또다른 엔트리의 식별자입니다.그리고 이 식별자는 유일해야합니다.

식별자로 사용되는 front matter는 다음 순서대로 입니다.

.Name > .LinkTitle > .Title
